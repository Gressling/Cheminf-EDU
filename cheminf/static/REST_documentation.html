REST API Documentation

GET /api/molecules

Summary: Get all molecules

Responses:

  • 200: List of molecules (array of Molecule objects)
  • 500: Internal server error

Example Request:

GET /api/molecules HTTP/1.1
Host: localhost:8050

Example Response (200):

[
  {
    "id": 1,
    "MoleculeUpacName": "Water"
  },
  {
    "id": 2,
    "MoleculeUpacName": "Ethanol"
  }
]

POST /api/molecules

Summary: Create a new molecule

Request Body: NewMolecule object

Responses:

  • 201: Molecule created successfully
  • 400: Invalid request payload
  • 500: Internal server error

Example Request:

POST /api/molecules HTTP/1.1
Content-Type: application/json

{
  "MoleculeUpacName": "Acetone"
}

Example Response (201):

{
  "id": 3,
  "MoleculeUpacName": "Acetone",
  "message": "Molecule created successfully"
}

PUT /api/molecules/{id}

Summary: Update a molecule

Path Parameter: id (integer)

Request Body: NewMolecule object

Responses:

  • 200: Molecule updated successfully
  • 400: Invalid request payload
  • 500: Internal server error

Example Request:

PUT /api/molecules/3 HTTP/1.1
Content-Type: application/json

{
  "MoleculeUpacName": "Isopropanol"
}

Example Response (200):

{
  "message": "Molecule updated successfully"
}

GET /api/inventory/search?q={InventoryNameSubstring}

Summary: Search inventory entries by molecule name (substring)

Query Parameter: q (string)

Responses:

  • 200: Search results (array of Inventory objects)
  • 500: Internal server error

Example Request:

GET /api/inventory/search?q=ethanol HTTP/1.1
Host: localhost:8050

Example Response (200):

[
  {
    "id": 2,
    "MoleculeUpacName": "Ethanol",
    "amount": 500,
    "unit": "ml"
  }
]
